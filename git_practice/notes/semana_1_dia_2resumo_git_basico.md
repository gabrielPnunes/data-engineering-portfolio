## Resumo - Fundamento de Git

 Este documento é um agrupamento das principais observações iniciais em relação ao Git

 Git é um sistema de controle de versões distribuídas, em resumo ele agrupa as versões de colabodores e
 suas em apenas um lugar e permite principalemnte as seguientes ações:

- Versionar Código(Versões anteriores)
- Manter Histórico de alteções
- Trabalhar em equipe de forma segura
- Evitar perda de código
