# Git Practice

## Obetivo

Este repositório foi criado com o objetivo de praticar e e **simular o uso do git hub em um fluxo real de trabalho**,
incluindo versionamento, branches, pull requests e resoluções de conflitos.

O foco principal será no processo de **colaboração e controle de versões**, a qual é uma habildade primordial para qualquer Engenheiro de Dados

aqui além de algumas anotações e observações ficará o controle de versões do principal repositório que são os código, python, sql etc.
