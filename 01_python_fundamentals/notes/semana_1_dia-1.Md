# Fundamentos de Python — Anotações Essenciais

## Tipos Básicos

Tipos definem **o que um dado representa** e **o que pode ser feito com ele**.

### Tipos mais usados

- `int` → números inteiros (ex: idade, contador)
- `float` → números decimais (ex: média, preço)
- `str` → texto (ex: nome, status)
- `bool` → verdadeiro ou falso (`True` / `False`)
- `list` → coleção ordenada e mutável
- `dict` → chave → valor

### Exemplos

idade = 25          # int
media = 7.5         # float
nome = "Gabriel"    # str
ativo = True        # bool
notas = [7, 8, 9]   # list
